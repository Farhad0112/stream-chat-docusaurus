The `TranslationContext` value contains the selected user language, as well as functions that allow for text and date translation. Exposes the `useTranslationContext` hook.

## Basic Usage

Pull values from context with our custom hook:

```jsx
const { t } = useTranslationContext();

<div className='message'>{t('This message will be translated.')}</div>;
```

## Values

### t

Translation function.

| Type     |
| -------- |
| function |

### tDateTimeParser

Function that parses time.

| Type     | Default |
| -------- | ------- |
| function | Dayjs   |

### userLanguage

String to set the user's language; (ex: 'en', 'fr', 'ru', etc).

| Type   | Default |
| ------ | ------- |
| string | 'en'    |
